## What is this?

A simple website written in [Go language](http://www.golang.org/) using the [Revel framework](http://revel.github.io/)

## Getting Started

A high-productivity web framework for the [Go language](http://www.golang.org/).

### Start the web server:

    revel run myapp

   Run with <tt>--help</tt> for options.
